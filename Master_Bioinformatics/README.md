# Master in Bioinformatics - UAB

Repository with the material corresponding to the  __Master in Bioinformatics__ given at 
[UAB](http://mscbioinformatics.uab.cat/base/base3.asp?sitio=msbioinformatics). Slides illustrating the use of R/Bioconductor packages, R code and datasets are available at [Biocloud Virtual Machines](http://www.biocloudservices.com/) provided to the students. This repository contains slides illustrating theoretical aspects, supplementary material, exercises and R code providing the answer to proposed tasks.
